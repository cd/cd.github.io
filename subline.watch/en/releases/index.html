Releases

Release notes of all SubLine Watch versions

v2.0.0

Released on January 3, 2023

  • The kiosk mode has been improved.
  • The machine assignments have been improved.
  • The user can now switch the vibration and the sound on and off separately for alerts.
  • If alarms occur simultaneously, the user is only alerted once.
  • The data point for the data point preview in the machine overview is now freely selectable.
  • The UX has been generally improved.
  • The configuration and commissioning has been made more consistent and easier. The administration tool has been removed.
  • The stats have been removed to increase the focus on the main functionality.
  • Various small improvements and bug fixes have been implemented.

v1.4.0-0

Released on May 23, 2022

  • The unequal operator for alarm conditions has been added.
  • The alarm conditions have been improved.
  • The access to the system time settings is now enabled.
  • A demo mode has been added.
  • Alarm volume and vibration are now adjustable.
  • Various small improvements and bug fixes have been implemented.

v1.3.0-0

Released on April 22, 2022

  • Users can assign themselves alarm messages or machines (more information).
  • PIN entry is required to change the connection settings.
  • The kiosk mode has been improved.
  • With a TLS-encrypted connection to the MQTT broker, the hostname verification or even the complete verification of the certificate can be deactivated.
  • Machine-specific, it can be set whether only the most important alarm message or all alarm messages should be displayed.
  • The "Alerts" key figure has been removed from the app.
  • A security pop-up has been added in the admin tool when updating the software.
  • Communication between the smartwatch app and the admin tool has been improved.
  • Importing licenses is no longer possible with the admin tool.
  • The validation of entries in the admin tool has been improved.
  • The display of the device online status in the admin tool has been improved.
  • JSON files from the admin tool are saved indented to make the files easier to read and edit.
  • Various small improvements and bug fixes have also been implemented.

v1.2.0-1

Released on March 11, 2022

  • A persistent MQTT connection is now the current recommendation of the admin tool.
  • Various small improvements and bug fixes have been implemented.

v1.2.0-0

Released on March 10, 2022

  • An "Always On" screen with additional information has been added.
  • The install and uninstall button has been removed from the admin tool
  • Various small improvements and bug fixes have been implemented.

v1.1.0-2

Released on February 18, 2022

  • The SubLine Watch Administrator Tool (admin tool) has been integrated into the "SubLine Watch" product.

v1.1.0

Released on October 27, 2021

  • An MQTT payload of format type JSON can now be processed.
  • Added new key figures: Operation Time, Availability, Offline Time, and Alerts
  • App exit and autostart behavior has been changed.
  • The logging has been improved.
  • The MQTT-based license check has been removed.
  • The app is now more robust against disconnections and faulty MQTT payload.
  • Various minor improvements and bug fixes have also been implemented.

v1.0.0

Released on September 16, 2021
