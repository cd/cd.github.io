Some of my projects

  • Responsler is a SaaS and my latest project. I built it to help developers better understand and debug HTTP communication.
  • SubLine Watch is a smartwatch solution that I created for the manufacturing industry. It is a conventional smartwatch for end users, running my software, which turns the device into a single-purpose device that supports workers in their daily work.
  • Whiskey Kinetics is a simple and realistic 2D physics engine I developed in JavaScript that runs in a browser. However, more than a PoC didn't come out of it, because I stopped the development at a very early stage.
  • Lynch is a JavaScript framework that I developed as a minimalistic alternative to established and bloated front-end frameworks. By now, the project is outdated and doesn't really have a right to exist anymore.
  • Liga Guru is an AI that I developed to predict soccer results. It supports betting on sports events of the German Bundesliga by identifying irrational high odds of betting providers.
  • Crillance was a fast-paced breakout logic game for Android smartphones that was a remake of the C64 classic Crillion. Unfortunately, it is no longer available in the PlayStore. Only a gameplay video from 2015 still exists.
